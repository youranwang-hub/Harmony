# 实验5-2 呼吸灯

本实验使用 TIM3 通道 2 输出 PWM，通过改变比较值控制 LED1 亮度，并用按键调节亮度。

## 文件说明

| 文件 | 功能 |
| --- | --- |
| `main.c` | 配置 TIM3_CH2 PWM 输出，KEY1 增加 PWM 比较值，KEY2 减少 PWM 比较值 |

## 硬件连接

| 外设 | 引脚 | 说明 |
| --- | --- | --- |
| LED1 / TIM3_CH2 | PB5 | 复用为 TIM3 通道 2 PWM 输出，LED 低电平点亮 |
| KEY1 | PA0 | 上拉输入，按下为低电平，用于增加亮度 |
| KEY2 | PC13 | 上拉输入，按下为低电平，用于降低亮度 |

## 程序逻辑

程序主要流程：

1. 使能 GPIOA 和 GPIOC 时钟。
2. 将 PA0、PC13 配置为上拉输入。
3. 使能 TIM3 和 GPIOB 时钟。
4. 将 PB5 映射到 `GPIO_AF_TIM3`，作为 TIM3_CH2 复用输出。
5. 设置 TIM3 预分频值 `TIM_Prescaler = 84`。
6. 设置自动重装载值 `TIM_Period = 500 - 1`。
7. 配置输出比较模式为 `TIM_OCMode_PWM1`。
8. 配置输出极性为 `TIM_OCPolarity_Low`，符合 LED 低电平点亮的硬件逻辑。
9. 启动 TIM3。

程序启动后，先依次写入 499、300、100、0 四个比较值，每个状态保持 500 ms，用于测试 LED 不同亮度。

## 按键控制

自动测试结束后进入主循环：

| 按键 | 程序动作 | 现象 |
| --- | --- | --- |
| KEY1 / PA0 | `PWM_Value++`，并写入 `TIM_SetCompare2()` | LED 亮度逐渐增加 |
| KEY2 / PC13 | `PWM_Value--`，并写入 `TIM_SetCompare2()` | LED 亮度逐渐降低 |

主循环每 20 ms 扫描一次按键。代码中 `PWM_Value` 的范围控制为 0 到 500，定时器 ARR 为 499，因此 500 可视为最高亮度端点。

## 运行现象

下载运行后，LED1 会先自动显示几档亮度变化，然后进入按键调光模式。按住 KEY1 时亮度逐步增强，按住 KEY2 时亮度逐步减弱。

## 使用方法

将 `main.c` 加入 STM32F407 标准外设库工程，编译下载后通过 KEY1 和 KEY2 调节 LED1 亮度。
