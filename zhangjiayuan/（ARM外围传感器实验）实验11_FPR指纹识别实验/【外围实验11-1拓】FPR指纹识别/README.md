# 实验11-1拓 FPR指纹识别拓展实验

## 实验内容

本拓展实验在指纹录入和匹配基础上，增加匹配分数显示、指纹库容量统计以及按 ID 删除指纹模板功能。程序仍通过 USART2 提供菜单交互，便于在实验过程中管理指纹库。

## 代码结构

- `代码/USER/main.c`：拓展主程序，显示四项菜单并处理串口输入。
- `代码/BSP/fpr.c`、`代码/BSP/fpr.h`：ZN632 通信协议、指纹库索引、录入、匹配、删除和容量统计。
- `代码/BSP/usart.c`、`代码/BSP/usart.h`：USART2 调试输出。

## 与基础版的区别

- 菜单由 2 项扩展为 4 项。
- 指纹匹配成功后输出指纹 ID 和匹配分数。
- 新增指纹库信息显示，统计已用数量、空闲数量和总容量。
- 新增按 ID 删除指纹模板，删除前会检查 ID 范围和该位置是否已有指纹。

## 菜单功能

```text
1 Add New Finger
2 Match Finger
3 Show Finger Library
4 Delete Finger By ID
```

- `1`：录入新指纹。
- `2`：匹配指纹，并显示匹配 ID 与 Score。
- `3`：读取索引表，显示指纹库使用情况。
- `4`：输入模板 ID，删除指定位置的指纹。

## 程序流程

1. 初始化 USART2 和 ZN632 指纹模块。
2. 校验密码并读取指纹库索引表。
3. 调用 `FPR_GetMenuChoice()` 获取用户输入，并过滤回车换行。
4. 根据菜单选项调用录入、匹配、库信息显示或删除函数。
5. 删除成功后更新本地索引表状态。

## 运行现象

匹配成功时串口会输出类似 `Finger Match, ID:x, Score:y` 的结果。选择库信息可看到已录入数量和剩余容量。选择删除功能后，输入有效且已占用的 ID，程序会调用模块删除命令并提示删除结果。

## 注意事项

- 当前代码中指纹库索引最大值按 `ZN632_INDEX_MAX 240` 处理。
- 删除前请确认 ID，删除后该模板不可恢复，需要重新录入。
- 指纹模块与 NFC 模块共用通信资源时，实验前务必切换到指纹模块侧。
