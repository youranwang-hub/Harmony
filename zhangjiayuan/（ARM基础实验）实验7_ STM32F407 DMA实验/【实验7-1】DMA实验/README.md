# 实验7-1 DMA 串口发送

## 1. 实验目标

本实验属于 **DMA 数据传输实验**，当前版本为 **基础/拓展**。实验围绕 STM32F407 标准外设库展开，重点训练单个模块从初始化、运行到结果验证的完整流程。

- 掌握 DMA 控制器基本配置。
- 理解内存到外设的数据搬运过程。
- 使用 DMA 减少 USART 逐字节发送负担。

## 9. 基础版与拓展版差异

拓展内容可在基础 DMA 发送基础上增加更长数据、重复发送或发送完成提示，重点比较普通串口发送与 DMA 发送的差异。

## 10. 常见问题与调试建议

| 问题 | 可能原因与处理方法 |
|---|---|
| 串口无输出 | 检查 DMA 通道、USART 地址和 TX 使能。 |
| 只发一部分 | 检查传输长度和完成标志。 |
| 下载成功但无现象 | 先确认程序是否进入 `main()`，再用 LED 最小程序判断板子和下载配置是否正常。 |
| 编译报头文件找不到 | 检查 Keil Include Path 是否包含 `USER`、`BSP`、`UTILS`、`CORE`、`FWLIB/inc` 等目录。 |
| 现象与预期相反 | 检查 LED、蜂鸣器、按键或模块信号是否为低电平有效。 |
