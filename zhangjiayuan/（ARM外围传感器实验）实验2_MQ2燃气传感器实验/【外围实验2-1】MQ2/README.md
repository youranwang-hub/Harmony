# MQ2 燃气传感器基础实验说明

本实验对应 `【外围实验2-1】MQ2`，代码位于 `代码` 文件夹中。程序通过 ADC1 通道 12 采集 MQ2 模拟输出，并通过 USART2 输出 ADC 值和换算电压。

## 实验目的

- 掌握 STM32F407 ADC1 的基础配置流程。
- 学会将 PC2 配置为模拟输入，对应 ADC1_CH12。
- 理解 12 位 ADC 数值与输入电压之间的换算关系。
- 通过串口观察 MQ2 输出随环境变化的相对变化。

## 硬件连接

| 模块 | 引脚/资源 | 说明 |
| --- | --- | --- |
| MQ2 AO | PC2 / ADC1_CH12 | 模拟输出接 ADC 输入 |
| MQ2 VCC | 开发板电源 | 按模块要求接入 |
| MQ2 GND | GND | 与开发板共地 |
| USART2_TX/RX | PA2 / PA3 | 串口输出 ADC 和电压 |

## 代码结构

```text
代码
├─BSP
│  ├─mq.c / mq.h             # MQ2 ADC 初始化与读取
│  └─usart.c / usart.h       # USART2 与 printf 重定向
├─SYSTEM
│  └─delay.c / delay.h       # 延时函数
└─USER
   └─main.c                  # 主程序：读取 ADC 并输出电压
```

## 核心流程

1. 初始化 USART2，用于串口输出。
2. 初始化 MQ2 ADC 采集模块。
3. 将 PC2 配置为模拟输入。
4. 配置 ADC1 规则通道 12。
5. 主循环中每 1 s 启动一次 ADC 软件转换。
6. 读取 12 位 ADC 值并换算为电压。
7. 通过串口输出 `AD=xxxx, x.xxxV`。

## 运行现象

正常环境下 ADC 值围绕一定范围波动。向 MQ2 金属网罩附近施加气体扰动后，ADC 值会上升；停止扰动后逐步回落。实验报告中正常环境读数主要为 292~305，对应电压约 0.235~0.246 V，扰动后曾观察到 509。

## 串口示例

```text
MQ Init OK
MQ2 ADC Test Start
AD=0292, 0.235V
AD=0305, 0.246V
AD=0509, 0.410V
```

## 注意事项

- MQ2 未经标定时，ADC 值只能表示相对变化，不能直接换算为 ppm 浓度。
- 传感器需要一定预热时间，刚上电时读数可能不稳定。
- ADC 波动受供电、参考电压、传感器噪声和环境气流影响。
- 如果串口无输出，应检查 `usart.c` 是否加入工程并完成 `printf` 重定向。
