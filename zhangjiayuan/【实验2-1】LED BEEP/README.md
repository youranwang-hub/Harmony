# 实验2-1 LED 与蜂鸣器控制

## 1. 实验目标

本实验属于 **GPIO 基础输出实验**，当前版本为 **基础版**。实验围绕 STM32F407 标准外设库展开，重点训练单个模块从初始化、运行到结果验证的完整流程。

- 掌握 GPIO 输出模式配置。
- 理解 LED 与蜂鸣器的有效电平。
- 能够通过程序控制板载 LED 亮灭和蜂鸣器鸣叫。

1. 使用 Keil uVision5 打开本实验目录中的工程文件。
2. 确认芯片型号为 STM32F407VE，并使用 STM32F4 标准外设库。

## 10. 常见问题与调试建议

| 问题 | 可能原因与处理方法 |
|---|---|
| 复位后蜂鸣器一直响 | 检查蜂鸣器是否低电平有效，初始化末尾应设置为关闭电平。 |
| LED 没反应 | 检查 GPIO 端口、引脚和板载 LED 标注是否一致。 |
| 下载成功但无现象 | 先确认程序是否进入 `main()`，再用 LED 最小程序判断板子和下载配置是否正常。 |
| 编译报头文件找不到 | 检查 Keil Include Path 是否包含 `USER`、`BSP`、`UTILS`、`CORE`、`FWLIB/inc` 等目录。 |
| 现象与预期相反 | 检查 LED、蜂鸣器、按键或模块信号是否为低电平有效。 |
