# STM32F407 RTC 基础实验说明

本实验对应 `【实验8-1】RTC`，代码位于 `代码` 文件夹中。实验通过 STM32F407 内部 RTC 获取时间和日期，并使用 OLED 显示。

## 实验目的

- 掌握 STM32F407 内部 RTC 的初始化方法。
- 理解 RTC 备份寄存器用于判断是否首次配置的作用。
- 学会读取 RTC 时间、日期并格式化显示。
- 熟悉 OLED 显示驱动在嵌入式实验中的使用方式。

## 硬件连接

| 模块 | 引脚/资源 | 说明 |
| --- | --- | --- |
| OLED_SCLK | PB6 | OLED 软件 IIC 时钟控制线 |
| OLED_SDIN | PB7 | OLED 软件 IIC 数据控制线 |
| RTC | STM32F407 内部 RTC | 使用 RTC 时钟和备份寄存器保存初始化标志 |

## 代码结构

```text
代码
├─BSP
│  ├─delay.c / delay.h       # 延时函数
│  ├─oled.c / oled.h         # OLED 显示驱动
│  ├─oledfont.h              # OLED 字库
│  └─rtc.c / rtc.h           # RTC 初始化、时间日期设置
└─USER
   └─main.c                  # 主程序：RTC 读取与 OLED 显示
```

## 核心流程

1. 延时等待 OLED 上电稳定。
2. 初始化 OLED 并清屏。
3. 调用 `BSP_RTC_Init()` 初始化 RTC。
4. 如果备份寄存器 `RTC_BKP_DR0` 中没有初始化标志，则设置默认时间和日期。
5. 主循环中读取 RTC 时间并显示为 `Time:HH:MM:SS`。
6. 主循环中读取 RTC 日期并显示为 `Date:20YY-MM-DD`。
7. 每约 400 ms 刷新一次 OLED 页面。

## 运行现象

OLED 上显示当前时间和日期，时间会持续递增。若 RTC 已经初始化过，程序会直接读取当前 RTC；若未初始化，则使用代码中的默认时间日期。

## 关键参数

| 参数 | 值 |
| --- | --- |
| 时间格式 | 24 小时制 |
| 显示设备 | 128x64 OLED |
| 默认时间 | 13:00:00 |
| 默认日期 | 2021-12-15 |
| 刷新周期 | 约 400 ms |

## 注意事项

- OLED 显示异常时，优先检查 PB6、PB7 连接和 OLED 供电。
- RTC 首次运行会写入默认时间；若需要重新初始化，可清除备份寄存器或修改初始化判断逻辑。
- 若 RTC 走时异常，应检查 RTC 时钟源和工程中标准外设库配置。
