# 实验5-1 时钟定时

本实验使用 TIM3 的更新中断控制 LED1 周期性亮灭，验证通用定时器的基本定时功能。

## 文件说明

| 文件 | 功能 |
| --- | --- |
| `main.c` | 配置 TIM3 时基单元和更新中断，在中断服务函数中翻转 LED1 状态 |

## 硬件连接

| 外设 | 引脚 | 有效电平 |
| --- | --- | --- |
| LED1 | PB5 | 低电平点亮 |

## 程序逻辑

程序主要流程：

1. 使能 GPIOB 时钟。
2. 将 PB5 配置为普通推挽输出。
3. 初始化时将 PB5 置高，使 LED1 熄灭。
4. 使能 TIM3 外设时钟。
5. 设置 TIM3 为向上计数模式。
6. 设置自动重装载值 `TIM_Period = 4999`。
7. 设置预分频值 `TIM_Prescaler = 8400`。
8. 清除 TIM3 更新标志，避免刚启动就进入中断。
9. 使能 TIM3 更新中断。
10. 配置 NVIC 中的 `TIM3_IRQn`。
11. 启动 TIM3。

主循环 `while(1)` 中没有具体任务，LED 状态完全由 `TIM3_IRQHandler()` 控制。

## 中断处理

`TIM3_IRQHandler()` 中使用静态变量 `i` 记录上一次状态：

- `i` 为 1 时，输出低电平，点亮 LED1。
- `i` 为 0 时，输出高电平，熄灭 LED1。
- 每次处理完后调用 `TIM_ClearITPendingBit()` 清除更新中断挂起标志。

## 运行现象

程序下载运行后，LED1 会周期性亮灭。由于每次 TIM3 更新中断都会翻转一次 PB5 输出电平，所以完整的亮灭周期由两次更新中断组成。

## 使用方法

将 `main.c` 加入 STM32F407 标准外设库工程，编译下载后观察 PB5 对应 LED1 的闪烁现象。
