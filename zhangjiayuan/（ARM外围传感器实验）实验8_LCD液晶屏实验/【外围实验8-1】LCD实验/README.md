# 实验8-1 LCD液晶屏实验

## 实验内容

本实验使用 STM32F407 通过 FSMC 并口驱动 3.2 寸 TFT-LCD 液晶屏，LCD 控制器为 ILI9341。当前代码完成 LCD 初始化、背光控制、清屏、英文字符字体设置和字符串显示，用于验证 LCD 8080 并口通信和基础显示功能。

## 代码结构

- `代码/USER/main.c`：实验主程序，初始化串口和 LCD，并显示固定文本。
- `代码/BSP/lcd_ili9341.c`、`代码/BSP/lcd_ili9341.h`：ILI9341 初始化、寄存器读写、清屏、颜色和字符显示接口。
- `代码/BSP/font.c`、`代码/BSP/font.h`：ASCII 字模数据。
- `代码/BSP/usart.c`、`代码/BSP/usart.h`：USART2 调试输出。

## 硬件与接口

- LCD：3.2 寸 TFT-LCD，分辨率 240 x 320
- LCD 控制器：ILI9341
- 通信方式：FSMC 模拟 8080 并口
- LCD 命令地址：`0x60000000`
- LCD 数据地址：`0x60020000`
- 背光控制：PD12
- 复位控制：PE1
- 调试串口：USART2，波特率 115200

## 程序流程

1. 初始化系统延时和 USART2。
2. 串口打印 `LCD TEST`。
3. 调用 `LCD_Init()` 初始化 ILI9341。
4. 使用 `LCD_SetFontEN(&ASCII_8x16)` 设置英文字体。
5. 设置前景色为红色、背景色为黑色，并清空全屏。
6. 在屏幕前两行显示：

```text
3.2 inch LCD:
Image resolution:240x320 px
```

## 运行现象

程序下载后，LCD 背光点亮，屏幕清为黑色背景，并以红色字体显示 LCD 尺寸和分辨率信息；串口终端同步输出实验启动信息。

## 注意事项

- 若屏幕无显示，优先检查 FSMC 引脚、LCD 插接方向、背光 PD12 和复位 PE1。
- 当前实验只验证英文字符显示，未涉及中文点阵和图片显示。
- 屏幕坐标与颜色接口均封装在 `lcd_ili9341` 驱动中，后续拓展可直接调用显示函数。
