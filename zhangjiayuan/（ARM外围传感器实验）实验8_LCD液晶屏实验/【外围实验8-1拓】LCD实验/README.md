# 实验8-1拓 LCD液晶屏拓展实验

## 实验内容

本拓展实验在 LCD 基础显示功能上增加动态界面效果：背景颜色周期切换、计数值动态刷新、彩色方块左右移动并在边界反弹。代码通过固定刷新节奏展示 LCD 局部清除、颜色填充、字符串刷新和简单动画控制。

## 代码结构

- `代码/USER/main.c`：拓展主程序，实现背景切换、计数器和移动方块动画。
- `代码/USER/main(拓展2：交互拓展）.c`：目录中额外保留的拓展源文件，当前 README 主要对应默认 `main.c`。
- `代码/BSP/lcd_ili9341.c`、`代码/BSP/lcd_ili9341.h`：LCD 初始化、清屏、块填充和字符串显示。
- `代码/BSP/font.c`、`代码/BSP/font.h`：ASCII 字体。
- `代码/BSP/usart.c`、`代码/BSP/usart.h`：USART2 调试输出。

## 与基础版的区别

- 基础版显示固定文字，拓展版显示持续刷新的动态图形界面。
- 每 1 秒更新一次计数值。
- 每 3 秒切换一次背景颜色，颜色包括红、绿、蓝、白、黑。
- 屏幕中部绘制 20 x 20 的彩色方块，并以固定步长左右移动。
- 方块到达屏幕左右边界后自动反向，实现简单动画效果。

## 关键参数

```text
BLOCK_SIZE       20
BLOCK_Y          100
BLOCK_STEP       2
FRAME_DELAY_MS   30
COUNT_UPDATE_MS  1000
COLOR_CHANGE_MS  3000
```

这些宏定义集中控制方块尺寸、移动速度、刷新周期和颜色切换周期。

## 程序流程

1. 初始化延时、USART2 和 LCD。
2. 设置 ASCII 字体并绘制初始界面。
3. 在主循环中擦除旧方块、计算新坐标、绘制新方块。
4. 按时间间隔刷新计数值。
5. 按时间间隔切换背景颜色并重绘界面。

## 运行现象

LCD 会显示实验标题、当前背景颜色、计数值以及一个不断左右移动的彩色方块。计数值逐秒增加，背景颜色每隔约 3 秒切换一次，串口终端打印拓展测试启动信息。

## 注意事项

- 动画通过循环延时刷新实现，适合教学演示；若后续需要更稳定的刷新节奏，可改为定时器驱动。
- 当前默认 `main.c` 主要实现颜色、计数和移动方块，未在该文件中实现串口输入内容直接显示到 LCD。
