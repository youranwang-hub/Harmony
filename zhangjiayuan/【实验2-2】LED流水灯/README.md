# 实验2-2 LED 流水灯

本实验在实验2-1 GPIO 输出的基础上加入软件延时，实现三盏 LED 的流水显示。

## 文件说明

| 文件 | 功能 |
| --- | --- |
| `main.c` | 三盏 LED 按 LED1 -> LED2 -> LED3 顺序循环点亮 |
| `mainplus2.c` | 拓展实验，三盏 LED 按 LED3 -> LED2 -> LED1 顺序循环点亮，并在 LED1 点亮时启动蜂鸣器 |

## 硬件连接

| 外设 | 引脚 | 有效电平 |
| --- | --- | --- |
| LED1 | PB5 | 低电平点亮 |
| LED2 | PB0 | 低电平点亮 |
| LED3 | PB1 | 低电平点亮 |
| 蜂鸣器 | PA8 | 低电平响，仅 `mainplus2.c` 使用 |

## 程序逻辑

`main.c` 的主要流程：

1. 使能 GPIOB 时钟。
2. 将 PB5、PB0、PB1 配置为推挽输出。
3. 初始化时将三个 LED 对应引脚置高，使 LED 熄灭。
4. 在主循环中依次点亮 LED1、LED2、LED3。
5. 每次切换后调用 `delay_ms(500)`，使每个状态保持约 0.5 秒。

`mainplus2.c` 的主要流程：

1. 初始化 PB5、PB0、PB1 作为 LED 输出。
2. 初始化 PA8 作为蜂鸣器输出。
3. 主循环按 LED3、LED2、LED1 的顺序显示流水灯。
4. LED3 和 LED2 点亮时蜂鸣器关闭。
5. LED1 点亮时 PA8 输出低电平，蜂鸣器发声。

## 运行现象

运行 `main.c` 时，三盏 LED 从 LED1 到 LED3 依次点亮，形成向一个方向移动的流水灯效果。

运行 `mainplus2.c` 时，流水方向变为 LED3 到 LED1，并且当流水到 LED1 时蜂鸣器响一下，形成声光联动效果。

## 使用方法

将需要测试的源码文件替换为工程中的 `main.c`，在 Keil MDK 中编译并下载到 STM32F407 开发板运行。
