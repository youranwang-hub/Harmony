# Harmony

> 软件学院小学期 — STM32F407 完整实验项目

**开发板**：青软遨游100 QST-AU100  
**MCU**：STM32F407（ARM Cortex-M4，168MHz）  
**开发环境**：MDK 5.25 + STM32F4xx_DSP_StdPeriph_Lib_V1.4.0  
**库类型**：ST 标准外设库 (StdPeriph_Lib)

---

## 实验目录

### 基础外设系列

| 编号 | 实验名称 | 说明 |
|:----:|----------|------|
| 2-1 | LED BEEF | GPIO 输出控制 LED 与蜂鸣器 |
| 2-2 | LED 流水灯 | 软件延时实现流水灯效果 |
| 2-3 | 按键检测 | GPIO 输入读取按键，软件消抖 |
| 3-1 | 外部中断 | 按键调节流水灯速度 |
| 4-1 | 串行通信 | USART2 中断接收，串口指令控制外设 |
| 5-1 | 时钟和定时器 | TIM3 定时中断，多频 LED 闪烁 |
| 5-2 | PWM 呼吸灯 | TIM3 PWM 输出，按键调亮度 |
| 6-1 | ADC 电位器 | ADC1 采样电位器，控制 LED 亮度/闪烁频率 |
| 6-2 | DAC 电压输出 | DAC_OUT1 输出锯齿波 |
| 7-1 | DMA 实验 | DMA1 搬运数据到 USART2，CPU 零干预 |
| 8-1 | RTC 时钟 | LSE 32.768kHz + OLED 显示年月日时分秒 |
