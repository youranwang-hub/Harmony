# Harmony
软件学院小学期STM32F407完整实验项目
开发板：青软遨游100 QST-AU100
开发环境：MDK5.25 + STM32F4xx_DSP_StdPeriph_Lib_V1.4.0

## 实验目录
1. 【实验2-1】LED BEEF：GPIO基础输出，点亮LED
2. 【实验2-2】LED流水灯：循环切换多LED实现流水效果
3. 【实验2-3】按键检测：GPIO输入轮询读取按键电平
4. 【实验3-1】外部中断：PA0下降沿外部中断，按键修改流水灯闪烁频率
