# Harmony

> 软件学院小学期 — STM32F407 实验项目（团队协作仓库）

**开发板**：青软遨游100 QST-AU100  
**MCU**：STM32F407（ARM Cortex-M4，168MHz）  
**开发环境**：MDK 5.25 + STM32F4xx_StdPeriph_Lib V1.4.0

## 实验范围

### 基础外设

| 编号 | 实验名称 | 一句话说明 |
|:---:|----------|------------|
| 2-1 | LED BEEF | GPIO 推挽输出控制 LED 亮灭与蜂鸣器开关 |
| 2-2 | LED 流水灯 | 软件空循环延时实现流水灯效果 |
| 2-3 | 按键检测 | GPIO 上拉输入读取按键状态，软件 20ms 消抖 |
| 3-1 | 外部中断 | EXTI 中断响应按键，动态调节流水灯速度 |
| 4-1 | 串行通信 | USART2 中断接收 + 串口指令控制 LED 与蜂鸣器 |
| 5-1 | 时钟和定时器 | 配置系统时钟 168MHz，TIM3 定时中断实现多频 LED 闪烁 |
| 5-2 | PWM 呼吸灯 | TIM3 CH2 输出 PWM，按键调节占空比改变 LED 亮度 |
| 6-1 | ADC 电位器 | ADC1 12-bit 采样电位器电压，控制 LED 亮度与闪烁频率 |
| 6-2 | DAC 电压输出 | DAC_OUT1 输出锯齿波，万用表/示波器观测模拟电压 |
| 7-1 | DMA 实验 | DMA1 Stream6 搬运数据到 USART2，CPU 零干预传输 |
| 8-1 | RTC 时钟 | LSE 32.768kHz 驱动 RTC，OLED 显示年月日时分秒 |

### 传感器

| 编号 | 实验名称 | 一句话说明 |
|:---:|----------|------------|
| 1 | DHT11 温湿度 | 单总线协议驱动 DHT11，采集温湿度并串口打印 |
| 2 | MQ2 燃气传感器 | ADC 采样 MQ2 模拟输出，检测可燃气体浓度 |

### 总线与协议

| 编号 | 实验名称 | 一句话说明 |
|:---:|----------|------------|
| 5 | I²C 与 E2PROM | 硬件 I2C1 驱动 AT24C02，实现跨页批量读写与自校验 |
| 6 | SPI 与 FLASH | 硬件 SPI1 驱动 W25Q64，扇区擦除 + 页写入 + JEDEC ID 识别 |
| 7 | CAN 总线自发自收 | bxCAN LoopBack 模式 500kbps 收发，验收过滤器接收所有报文 |
| 10 | GPS 定位 | UART4 接收 LC86L 模块数据，解析 NMEA 0183 六种语句 |

### 人机交互

| 编号 | 实验名称 | 一句话说明 |
|:---:|----------|------------|
| 3&4 | IR 红外收发 | NEC 协议 38kHz 载波编码发送 + TIM 输入捕获解码接收 |
| 8 | LCD 液晶屏 | FSMC 模拟 8080 并口驱动 ILI9341，8×16 ASCII 字模显示 |
| 9 | XPT 触摸屏 | 软件 SPI 驱动 XPT2046，去极值滤波 + 仿射变换坐标校准 |

### 综合应用

| 编号 | 实验名称 | 一句话说明 |
|:---:|----------|------------|
| 11 | FPR 指纹识别 | UART4 驱动 ZN632，自定义包协议实现指纹录入与 1:N 匹配 |
| 12 | NFC 读写卡 | UART4 驱动 PN532，HSU 帧协议对 Mifare S50 进行 16 字节读写 |
