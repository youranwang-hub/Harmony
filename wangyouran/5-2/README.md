# 实验 5-2：定时器 PWM 呼吸灯

## 实验目的

配置 TIM3 为 PWM 输出模式，通过 PB5 驱动 LED，实现脉宽调制改变 LED 亮度。通过两个按键增减 PWM 值，实现手动调节亮度的功能。

## 硬件连接

| 外设 | 引脚 | 说明 |
|------|------|------|
| LED1 | PB5 | TIM3_CH2，PWM 输出，低电平点亮 |
| KEY1 | PA0 | 按下：亮度增加 |
| KEY2 | PC13 | 按下：亮度降低 |

## PWM 配置

| 参数 | 值 | 计算 |
|------|-----|------|
| TIM3 时钟 | 84MHz | APB1=42MHz × 2 |
| TIM_Prescaler | 84 | 84MHz ÷ 84 = 1MHz |
| TIM_Period | 499 | 1MHz ÷ 500 = **2kHz** |
| PWM 模式 | PWM1 | CNT < CCR → 有效电平 |
| 输出极性 | Low | 有效电平 = 低（点亮 LED） |
| 输出通道 | CH2 | PB5 复用 |

### 占空比与亮度

| CCR 值 | 占空比 | LED 亮度 |
|--------|--------|----------|
| 499 | ≈100% | 最亮 |
| 300 | ≈60% | 较亮 |
| 100 | ≈20% | 微亮 |
| 0 | 0% | 灭 |

## 基础版 (basic/main.c)

### 功能

1. **自动演示**：上电后依次显示 4 档亮度（最亮 → 较亮 → 微亮 → 熄灭），每档 0.5s
2. **手动调节**：演示结束后 LED 停在半亮（250），用户通过按键调节

### 按键操作

| 按键 | 引脚 | 功能 | 步进 | 范围 |
|------|------|------|------|------|
| KEY1 | PA0 | 增亮 | +50/次 | 0 → 499 |
| KEY2 | PC13 | 变暗 | -50/次 | 499 → 0 |

每次按键后原地等待抬起，防止连按飙到极值。

### 代码修复记录

| # | 问题 | 修复 |
|---|------|------|
| 1 | 注释写 `CCR1` 实际是 `CCR2` | 改为 `CCR2` |
| 2 | 按键无抬起检测，按住连续跑 | 添加 `while` 等待抬起 |
| 3 | 步进 ±1 太小，肉眼不可见 | 改 ±50，10 次按到底 |
| 4 | 演示结束后 PWM=0，KEY2 无法变暗 | 初始化为 250（半亮） |

### 运行流程

```
上电 → 自动演示 4 档亮度 → LED 停在半亮
                                   │
                          ┌────────┴────────┐
                          ↓                  ↓
                     KEY1 增亮          KEY2 变暗
                     (+50/次)          (-50/次)
```
