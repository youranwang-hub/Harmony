# 实验 6-1：ADC 电位器控制 LED

## 硬件连接

| 外设 | 引脚 | 说明 |
|------|------|------|
| 电位器 | PC1 (ADC1_CH11) | 滑动变阻器输出 0~3.3V 模拟电压 |
| LED1 | PB5 | 低电平点亮 |

## ADC 配置参数

| 参数 | 值 | 说明 |
|------|-----|------|
| ADC 模块 | ADC1 | |
| 通道 | CH11 (PC1) | 电位器输入 |
| 分辨率 | 12bit | 0 ~ 4095 |
| 模式 | 独立模式、单次转换 | 软件触发 |
| ADC 时钟 | 14MHz | PCLK2=84MHz ÷ 6 |
| 采样周期 | 480 Cycles | |
| 均值次数 | 10 次 | `Get_Adc_Average()` |

## 信号链路

```
电位器(0~3.3V) → PC1 → ADC1_CH11 → 12bit转换(0~4095)
                                              ↓
                               basic: PWM_Value = 455 - Adc/9 → 调亮度
                            extended: Delay_Ms = 100 + Adc×1900/4095 → 调速
```

---

## 基础版（basic）：ADC 控制 LED 亮度（PWM）

### 功能

PWM 演示结束后，通过 ADC 读取电位器电压，将 ADC 值线性映射为 PWM 占空比，实现旋钮调亮度。

### PWM 参数（TIM3_CH2, PB5）

| 参数 | 值 |
|------|-----|
| TIM3 时钟 | 84MHz |
| 预分频 | 84 → 1MHz |
| Period | 499 → 2kHz PWM |
| 模式 | PWM1, 低电平有效 |

### ADC → PWM 映射

```
PWM_Value = 455 - Adc / 9
```

| 电位器位置 | ADC 值 | PWM_Value | 占空比 | LED 亮度 |
|-----------|--------|-----------|--------|----------|
| 最左 (0V) | ≈0 | 455 | ~91% | 最亮 |
| 中间 | ≈2048 | 227 | ~45% | 中等 |
| 最右 (3.3V) | ≈4095 | 0 | 0% | 熄灭 |

> 注意：PWM_Value 上限为 455 而非 499，原因是低电平有效时，CCR 值越大 LED 越亮。上限设为 455 防止 Adc 很小时 PWM_Value 超出 499 范围。

### 运行流程

1. 初始化 TIM3 PWM + ADC1
2. 演示 4 档亮度（最亮→较亮→微亮→灭），每档 0.5s
3. 进入 `while(1)`，每 100ms 读取 ADC 并更新 PWM

---

## 拓展版（extended）：ADC 控制 LED 闪烁频率（GPIO 翻转）

### 功能

不使用 PWM，直接通过 GPIO 翻转控制 LED 闪烁，ADC 值决定亮/灭保持时间。

### LED 配置

| 参数 | 值 |
|------|-----|
| 引脚 | PB5 |
| 模式 | 推挽输出 (OUT PP) |
| 控制方式 | `GPIO_ToggleBits` 翻转 |

### ADC → 延时映射

```
Delay_Ms = 100 + Adc × 1900 / 4095
```

| 电位器位置 | ADC 值 | 延时 | 闪烁效果 |
|-----------|--------|------|----------|
| 最左 (0V) | ≈0 | ~100ms | 快速闪烁 (~5Hz) |
| 中间 | ≈2048 | ~1050ms | 适中 |
| 最右 (3.3V) | ≈4095 | ~2000ms | 缓慢闪烁 (~0.25Hz) |

### 运行流程

```
初始化 LED(PB5 OUT) + ADC1
  ↓
while(1):
  读ADC(10次均值) → 算延时 → GPIO_ToggleBits → delay_ms(延时)
```

---

## 两版对比

| | basic（PWM 调亮度） | extended（翻转调速） |
|------|---------------------|----------------------|
| PB5 模式 | AF 复用 (TIM3_CH2) | OUT 推挽 |
| 使用 TIM3 | 是 | **否** |
| LED 控制 | `TIM_SetCompare2` 调占空比 | `GPIO_ToggleBits` 翻转 |
| 效果 | 拧电位器改变亮度 | 拧电位器改变闪烁频率 |
| 延时范围 | 固定 100ms 采样 | 100ms ~ 2000ms 动态 |
| 代码特点 | 非阻塞（PWM 硬件控制） | 阻塞（软件延时） |
