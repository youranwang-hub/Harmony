# 实验 3-1：按键调节流水灯速度

## 实验目的

掌握按键输入与流水灯输出结合的方法，通过检测按键状态动态改变流水灯速度。

## 硬件连接

| 外设  | GPIO 引脚 | 说明                       |
|-------|-----------|----------------------------|
| LED1  | PB5       | 推挽输出，低电平点亮          |
| LED2  | PB0       | 推挽输出，低电平点亮          |
| LED3  | PB1       | 推挽输出，低电平点亮          |
| KEY2  | PC13      | 上拉输入，按下 = 低电平(0)   |

## 版本说明

### 基础版 (`basic/main.c`)

固定速度流水灯（LED1 → LED2 → LED3），每步延时 300ms，循环执行。

### 拓展版 (`extended/main.c`)

通过 **KEY2** 动态调节流水速度：

| KEY2 状态 | 流水速度 | 说明    |
|-----------|---------|---------|
| 松开      | 300ms   | 默认快速 |
| 按住      | 1000ms  | 降速    |

**实现方式**：每轮循环先读取 KEY2，按住则 `delay_time = 1000`，松开则 `delay_time = 300`，然后三个 LED 共用这个变量进行延时。全程非阻塞，按住时流水灯仍正常跑动，不会卡死。

## 两版对比

| 项目       | 基础版     | 拓展版             |
|------------|-----------|-------------------|
| 流水速度    | 固定 300ms | 按键可切换 300/1000ms |
| 按键控制    | 无        | KEY2 按住减速        |

## 开发环境

- 芯片：STM32F4 系列
- 库：STM32F4 标准外设库
- IDE：Keil MDK (Vision)
