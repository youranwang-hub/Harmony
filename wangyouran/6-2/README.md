# 实验 6-2：DAC 输出电压实验

## 硬件连接

| 外设 | 引脚 | 说明 |
|------|------|------|
| DAC 输出 | PA4 (DAC_OUT1) | DAC 通道 1 模拟电压输出 |
| 测量 | PA4 ↔ GND | 使用万用表或示波器测量 |

## DAC 配置参数

| 参数 | 值 | 说明 |
|------|-----|------|
| DAC 模块 | DAC1, CH1 | |
| 输出引脚 | PA4 | 模拟模式 |
| 分辨率 | 12bit | 0 ~ 4095 |
| 对齐方式 | 右对齐 (`DAC_Align_12b_R`) | |
| 触发 | 无 (`DAC_Trigger_None`) | 软件写入即更新 |
| 波形发生器 | 关闭 | |
| 输出缓冲 | 关闭 (`OutputBuffer_Disable`) | |

## 电压计算公式

```
Vout = Vref × DOR / 4096 = 3.3V × DacValue / 4096
```

| DacValue | 输出电压 | 说明 |
|----------|----------|------|
| 0 | 0V | 最小值 |
| 2048 | ~1.65V | 中点 |
| 4095 | ~3.3V | 最大值 |

每步 ±248，电压变化：`248 / 4096 × 3.3V ≈ 0.2V`

---

## 基础版（basic）：递增锯齿波

### 功能

DAC 从 0V 开始，每秒递增约 0.2V，到达最大值（~3.3V）后跳回 0V，循环输出正向锯齿波。

### 逻辑

```
DacValue = 0 (初始)
  ↓
while(1):
  输出 DacValue → DacValue += 248 → 若 > 4095 则归零 → delay 1s
```

### 输出波形

```
3.3V ┤                                    ╱│
     │                                  ╱  │
     │                                ╱    │
     │                              ╱      │
     │                            ╱        │
  0V ┤──────────────────────────╱──────────┼──→ t
     0s                        17s        17s
```

> 周期 ≈ 4095 / 248 × 1s ≈ 17 秒

---

## 拓展版（extended）：递减反向锯齿波

### 功能

DAC 从最大值（~3.3V）开始，每秒递减约 0.2V，降到无法再减后跳回最大值，循环输出反向锯齿波。

### 逻辑

```
DacValue = 4095 (初始)
  ↓
while(1):
  输出 DacValue → 若 ≥248 则 DacValue -= 248，否则重置 4095 → delay 1s
```

### 边界安全

`DacValue` 为 `u16` 无符号类型，直接 `-= 248` 会导致下溢绕到 65535。代码采用**先判断再减**：

```c
if(DacValue >= 248)
    DacValue -= 248;   // 正常递减
else
    DacValue = 4095;   // 重置到最大值
```

### 输出波形

```
3.3V ┤╲
     │  ╲
     │    ╲
     │      ╲
     │        ╲
  0V ┤──────────╲──────────────────────────┼──→ t
     0s         17s                        17s
```

---

## 两版对比

| | basic（递增） | extended（递减） |
|------|-------------|-------------|
| 初始值 | `DacValue = 0` | `DacValue = 4095` |
| 变化方向 | `+= 248`（递增） | `-= 248`（递减） |
| 边界条件 | `> 4095` → 归零 | `< 248` → 重置 4095 |
| 波形 | 正向锯齿波 ↗ | 反向锯齿波 ↘ |
| 周期 | ~17s | ~17s |

## 观测方式

| 工具 | 操作 | 现象 |
|------|------|------|
| 万用表（电压档） | 红表笔接 PA4，黑表笔接 GND | 电压每秒跳变 0.2V |
| 示波器 | 探头接 PA4，地夹接 GND | 锯齿波，周期约 17s |
