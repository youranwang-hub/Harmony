# 实验 2-2：延时函数与流水灯控制

## 实验目的

掌握软件延时函数的编写与调用，通过延时结合 GPIO 输出实现流水灯效果，并实现 LED 与蜂鸣器的联动控制。

## 硬件连接

| 外设   | GPIO 引脚 | 说明               |
|--------|-----------|--------------------|
| LED1   | PB5       | 低电平点亮，高电平熄灭 |
| LED2   | PB0       | 低电平点亮，高电平熄灭 |
| LED3   | PB1       | 低电平点亮，高电平熄灭 |
| 蜂鸣器  | PA8       | 低电平响，高电平不响   |

## 核心方法

使用软件延时函数 `Delay_ms()` 控制每次灯亮/灭的持续时间，通过嵌套空循环实现毫秒级延时：

```c
void Delay_ms(uint32_t ms)
{
    uint32_t i, j;
    for (i = 0; i < ms; i++)
        for (j = 0; j < 10000; j++);
}
```

## 版本说明

### 基础版 (`basic/main.c`)

实现**向右流水灯**（LED1 → LED2 → LED3），每个 LED 亮 300ms 后熄灭，切换到下一个：

| 时刻 | LED1 | LED2 | LED3 |
|------|------|------|------|
| 1    | 亮   | 灭   | 灭   |
| 2    | 灭   | 亮   | 灭   |
| 3    | 灭   | 灭   | 亮   |

### 拓展版 (`extended/main.c`)

在基础版之上实现两个增强功能：

1. **向左流水灯**（LED3 → LED2 → LED1），与基础版方向相反
2. **LED1 亮时蜂鸣器发声**，LED2/LED3 亮时蜂鸣器不响

| 时刻 | LED1 | LED2 | LED3 | 蜂鸣器 |
|------|------|------|------|--------|
| 1    | 灭   | 灭   | 亮   | 不响    |
| 2    | 灭   | 亮   | 灭   | 不响    |
| 3    | 亮   | 灭   | 灭   | 响      |

## 两版对比

| 项目       | 基础版       | 拓展版           |
|------------|-------------|-----------------|
| 流水方向    | 向右（1→2→3） | 向左（3→2→1）    |
| 蜂鸣器控制  | 无           | LED1 亮时发声     |
| 延时        | 300ms       | 300ms            |
