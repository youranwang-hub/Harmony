# STM32F407 完整实验项目 — 王悠然

> 软件学院小学期 STM32F4 系列实验，涵盖基础外设驱动与综合应用模块。

**开发板**：青软遨游100 QST-AU100  
**MCU**：STM32F407（ARM Cortex-M4，168MHz）  
**开发环境**：MDK 5.25 + STM32F4xx_DSP_StdPeriph_Lib_V1.4.0  
**库类型**：ST 标准外设库（StdPeriph_Lib）

---

## 实验索引

### 第一部分：基础外设系列

从零开始掌握 STM32F4 的 GPIO、中断、串口、定时器、ADC/DAC、DMA、RTC 等核心外设。

| 编号 | 实验名称 | 说明 | 关键技术 |
|:---:|----------|------|----------|
| 2-1 | [LED BEEF](./【实验2-1】LED%20BEEF/) | GPIO 输出控制 LED 与蜂鸣器 | GPIO 推挽输出、高低电平控制 |
| 2-2 | [LED 流水灯](./【实验2-2】LED流水灯/) | 软件延时实现流水灯效果 | 软件空循环延时、GPIO 顺序控制 |
| 2-3 | [按键检测](./【实验2-3】按键检测/) | GPIO 输入读取按键，软件消抖 | GPIO 上拉输入、20ms 消抖、抬起检测 |
| 3-1 | [外部中断](./【实验3-1】外部中断/) | 按键调节流水灯速度 | 外部中断 EXTI、按键动态调速 |
| 4-1 | [串行通信](./【实验4-1】串行通信/) | USART2 中断接收，串口指令控制外设 | USART 配置、中断接收、printf 重定向、指令解析 |
| 5-1 | [时钟和定时器](./【实验5-1】时钟和定时器/) | TIM3 定时中断，多频 LED 闪烁 | 时钟树配置、PLL、定时器中断、NVIC 优先级 |
| 5-2 | [PWM 呼吸灯](./【实验5-2】定时器PWM呼吸灯功能实验/) | TIM3 PWM 输出，按键调亮度 | PWM 占空比调节、TIM 输出比较模式 |
| 6-1 | [ADC 电位器](./【实验6-1】ADC电位器/) | ADC1 采样电位器，控制 LED 亮度 | 12-bit ADC、模拟通道配置、扫描模式 |
| 6-2 | [DAC 电压输出](./【实验6-2】DAC电压输出/) | DAC_OUT1 输出锯齿波 | 12-bit DAC、双通道、定时器触发 |
| 7-1 | [DMA 实验](./【实验7-1】F407%20DMA实验/) | DMA1 搬运数据到 USART2，CPU 零干预 | DMA 流/通道映射、内存到外设传输模式 |
| 8-1 | [RTC 时钟](./【实验8-1】RTC实验/) | LSE 32.768kHz + OLED 年月日时分秒 | RTC 日历、LSE 晶振、OLED I²C 驱动 |

### 第二部分：传感器系列

掌握常用数字/模拟传感器的协议驱动与数据采集。

| 编号 | 实验名称 | 说明 | 关键技术 |
|:---:|----------|------|----------|
| 1 | [DHT11 温湿度传感器](./实验1_DHT11传感器实验/) | 单总线通信获取温湿度 | 单总线协议、GPIO 输入输出动态切换、微秒延时 |
| 2 | [MQ2 燃气传感器](./实验2_MQ2燃气传感器实验/) | ADC 读取可燃气体浓度 | 模拟传感器、ADC 采样、预热特性 |

### 第三部分：总线与协议系列

深入掌握 STM32 常用总线协议：I²C、SPI、CAN、UART（GPS/NMEA）。

| 编号 | 实验名称 | 说明 | 关键技术 |
|:---:|----------|------|----------|
| 5 | [I2C 与 E2PROM](./实验5_I2C与E2PROM实验/) | AT24C02 批量读写，写入-读出-校验 | 硬件 I2C1、开漏输出、页写入算法、ACK/NACK |
| 6 | [SPI 与 FLASH](./实验6_SPI与FLASH实验/) | W25Q64 扇区擦除 + 页写入 + 批量读 | 硬件 SPI1、Mode3、软件片选、JEDEC 识别 |
| 7 | [CAN 总线自发自收](./实验7_CAN总线自发自收实验/) | CAN LoopBack 模式收发测试 | bxCAN 外设、500kbps、验收过滤器、扩展帧 |
| 10 | [GPS 定位实验](./实验10_GPS定位实验/) | LC86L NMEA 0183 数据解析 | UART4 9600bps、IDLE 中断、GGA/RMC/GLL/GSA/GSV/VTG 解析 |

### 第四部分：人机交互与多媒体系列

LCD 显示、触摸输入、红外收发 — 构建完整人机交互系统。

| 编号 | 实验名称 | 说明 | 关键技术 |
|:---:|----------|------|----------|
| 3&4 | [IR 红外收发](./实验3&4_IR红外收发实验/) | NEC 协议 38kHz 载波编码发送与输入捕获解码 | TIM PWM 载波生成、TIM 输入捕获、NEC 时序 |
| 8 | [LCD 液晶屏](./实验8_LCD液晶屏实验/) | ILI9341 3.2 寸 TFT 字符显示 | FSMC 8080 并口、GRAM 窗口写、ASCII 字模 |
| 9 | [XPT 触摸屏](./实验9_XPT触摸屏实验/) | 电阻屏坐标采集与仿射变换校准 | 软件 SPI 模拟、去极值平均滤波、6 参数仿射变换 |

### 第五部分：综合应用系列

指纹识别、NFC 读写 — 结合串口协议栈的企业级应用场景实战。

| 编号 | 实验名称 | 说明 | 关键技术 |
|:---:|----------|------|----------|
| 11 | [FPR 指纹识别](./实验11_FPR指纹识别实验/) | ZN632 指纹录入与 1:N 匹配 | 自定义包协议、校验和、索引表位图、状态机消抖 |
| 12 | [NFC 读写卡](./实验12_NFC读写卡实验/) | PN532 对 Mifare S50 读写 16 字节 | HSU 帧协议、LCS/DCS 双重校验、ISO 14443A、Mifare 命令 |

```
├─ DMA 传输

                                      ├─ ZN632 指纹识别

```

## 目录规范

每个实验目录采用统一结构：

```
实验目录/
├── USER/
│   └── main.c              # 主程序入口
├── BSP/
│   ├── xxx.c / xxx.h       # 外设驱动（板级支持包）
│   └── usart.c / usart.h   # USART2 调试串口（fputc/fgetc 重定向）
└── UTILS/
    └── delay.c             # 软件延时函数（备用）
```
