# 实验 5-1：定时器中断控制 LED 闪烁

## 实验目的

通过配置 TIM3 通用定时器的中断功能，实现不同频率的 LED 闪烁。掌握 STM32F407 时钟系统结构、定时器参数配置以及 PLL 时钟参数对定时器的影响。

## 硬件连接

| 外设 | 引脚 | 说明 |
|------|------|------|
| LED1 | PB5 | 低电平点亮 |
| LED2 | PB0 | 低电平点亮 |
| LED3 | PB1 | 低电平点亮 |
| 蜂鸣器 | PA8 | 低电平发声 |
| TIM3 | 内部外设 | APB1 总线 |

## 时钟系统（STM32F407）

```
HSE(8MHz)
  │ ÷ PLL_M(8)
1MHz
  │ × PLL_N(336)
336MHz (VCO)
  │ ÷ PLL_P(2)
168MHz → SYSCLK
  │
  └→ AHB ÷1 → 168MHz
       └→ APB1 ÷4 → 42MHz
            └→ TIM3 ×2 → 84MHz  ← 定时器时钟
```

**关键规则**：当 APB1 预分频 ≠ 1 时，定时器时钟 = APB1 × 2。

## TIM3 配置与计算

| 参数 | 值 | 说明 |
|------|-----|------|
| 输入时钟 | 84MHz | APB1=42MHz × 2 |
| TIM_Prescaler | 8400 | 84MHz ÷ 8400 = 10kHz |
| TIM_Period | 4999 | 10kHz ÷ 5000 = 2Hz 中断 |
| 中断周期 | 0.5s | |

## 中断优先级

| 分组 | 抢占优先级 | 子优先级 |
|------|-----------|----------|
| NVIC_PriorityGroup_1 | 1 | 3 |

## 基础版 (basic/main.c)

### 功能

TIM3 每 0.5s 触发一次中断，中断服务函数中翻转 LED1(PB5)，实现 **1Hz 闪烁**（亮 0.5s + 灭 0.5s = 1s 一个周期）。

### 核心逻辑

```c
static u8 i;
if (i) { i = 0;  GPIO_ResetBits(GPIOB, GPIO_Pin_5); }  // 点亮
else   { i = 1;  GPIO_SetBits(GPIOB, GPIO_Pin_5);   }  // 熄灭
```

### 修复记录

| 问题 | 修复 |
|------|------|
| 未调用 NVIC_PriorityGroupConfig，默认 Group 0 不支持抢占优先级 1 | 添加 `NVIC_PriorityGroupConfig(NVIC_PriorityGroup_1)` |

---

## 拓展版 (extended/main.c)

### 功能

在基础版之上，用软件计数器 (`tick`) 驱动三个 LED 以**不同频率**同时闪烁，直观展示定时器中断的多种分频效果。

### LED 闪烁频率

| LED | 引脚 | 翻转条件 | 频率 | 视觉效果 |
|------|------|----------|------|----------|
| LED1 | PB5 | tick % 1 == 0 | 1Hz | 基准速度 |
| LED2 | PB0 | tick % 2 == 0 | 0.5Hz | 慢一倍 |
| LED3 | PB1 | tick % 4 == 0 | 0.25Hz | 慢四倍 |

蜂鸣器在每个周期起点 (tick=0) 短鸣一声作为标记。

### 时钟系统文档

拓展版代码头部包含完整的时钟树 ASCII 图和 PLL 修改指南，标明修改 `system_stm32f4xx.c` 中 `PLL_N` 的效果：

| PLL_N | SYSCLK | LED1 闪烁 | 效果 |
|-------|--------|-----------|------|
| 336 | 168MHz | 1Hz | 默认速度 |
| 168 | 84MHz | 0.5Hz | 全慢一半 |
| 336 | 168MHz | 1Hz | 默认速度 |

## PLL 参数修改实验

修改工程中的 `system_stm32f4xx.c`，改变 `PLL_N` 参数后重新编译烧录，观察三个 LED 闪烁速度的同步变化：

```
原始: #define PLL_N  336   →  LED快速闪烁
修改: #define PLL_N  168   →  LED闪烁慢一半（所有LED同比例变慢）
```

这验证了**系统时钟 → 定时器时钟 → 溢出中断 → LED 频率**的级联关系。
